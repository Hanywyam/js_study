JSON

JavaScript object notation

: 문자로 이루어진 데이터의 집합

  • 자료를 표현하는 방식
  • 데이터 교환 시 주로 사용
  • CSV, XML, CSON 등... 이 있지만 현재 웹에서 가장 많이 사용하는 방식
    1. {"key" : "value"} 형태로 작성
    2. 값value은 문자열, 숫자, 불린형만 가능(함수X)
    3. 키key에 반드시 큰 따옴표 필수
    4. 값value의 문자열도 반드시 큰 따옴표 필수
    5. 마지막 프로퍼티나 자료엔 객체와 달리 쉼표(,)를 쓰지 않음
